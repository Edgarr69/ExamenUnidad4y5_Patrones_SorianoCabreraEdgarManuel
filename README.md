# ExamenUnidad4y5_Patrones_SorianoCabreraEdgarManuel
# Soriano Cabrera Edgar Manuel
# Simulador de centro maritimo
# Descripcion General
Simulador Marítimo en C#: Proyecto de consola que simula el control de barcos usando patrones de diseño (Factory, Bridge, State) y arquitectura por capas con EventBus para manejo de eventos y historial en tiempo real.
# Ejecución

Requisitos previos

Descargar .NET 9.0 SDK desde https://dotnet.microsoft.com/download

Descargar Git desde https://git-scm.com/downloads

Paso 1: Clonar el repositorio

Abre Visual Studio Code o la terminal y ejecuta:

git clone https://github.com/Edgarr69/ExamenUnidad4y5_Patrones_SorianoCabreraEdgarManuel

Paso 2: Entrar a la carpeta del proyecto
cd ExamenUnidad4y5_Patrones_SorianoCabreraEdgarManuel

Paso 3: Compilar y ejecutar el programa

Si el proyecto es un archivo .cs:

csc Program.cs
Program.exe


O si el proyecto es de .NET:

dotnet build
dotnet run
